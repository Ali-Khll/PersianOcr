For running with server support you should flow these steps:
1-install nodejs (http://nodejs.org/)
2-close your cmd window and reopen it
3-in cmd go to the path that you extract this pakage
4-in cmd type `node server.js` (if that was not okay, enter this `setx PATH "%PATH%;C:\Program Files\nodejs"` and run that again)
5- in chrome browser open 127.0.0.1:8125
thats it!
